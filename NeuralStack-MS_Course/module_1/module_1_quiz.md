### Module 1 Quiz

1. Which security property is violated when a model reproduces sensitive data from its training set in its output?
   a) Availability
   b) Confidentiality
   c) Non-repudiation
   d) Integrity

2. What is the difference between training and fine-tuning?
   a) They are the same process
   b) Fine-tuning happens before training
   c) Fine-tuning is a smaller, secondary training pass to specialize an already-trained model
   d) Training only applies to vector databases

3. Which of the following is most accurate about where AI security incidents typically occur?
   a) Almost always inside the model's mathematical weights
   b) Most often in the data, deployment infrastructure, or application/agent layer
   c) Only in the training dataset
   d) Exclusively in the user interface

4. What is an embedding?
   a) A type of firewall rule
   b) A numerical vector representation of content used to measure semantic similarity
   c) A synonym for "weights"
   d) A method for encrypting model files

5. The context window refers to:
   a) The physical server running the model
   b) The amount of text a model can consider at once, including prompt and retrieved content
   c) The time window during which a model can be updated
   d) The number of users who can query a model simultaneously

**Answer key:** 1-b, 2-c, 3-b, 4-b, 5-b
