# AI Security Engineering for Beginners

A free, open-source introductory course on securing AI and LLM systems — built by **[NeuralStack | MS](https://neuralstackms.tech)**, a publication covering software engineering, AI engineering, and cybersecurity.

No prior security certification is required. Some familiarity with how machine learning models are built and deployed will help, but every technical term is defined the first time it appears.

---

## What you'll learn

By the end of this course, you will be able to:

- Explain why AI systems require a different threat model than traditional software
- Describe the ten risk categories in the OWASP Top 10 for LLM Applications and recognize them in real systems
- Identify where data poisoning, supply chain compromise, and provenance failures can occur in an ML pipeline
- Map the attack surface of a RAG (Retrieval-Augmented Generation) pipeline, from ingestion to generation
- Distinguish prompt injection from jailbreaking and describe baseline mitigations for each
- Speak the vocabulary of AI governance frameworks (NIST AI RMF, ISO/IEC 42001, MITRE ATLAS) well enough to navigate them
- Plan your next concrete steps into AI security engineering as a career path

## Who this is for

Engineers, analysts, and career-changers who understand how software systems work but are new to securing AI systems specifically — including anyone exploring a transition into AI security engineering as a career path.

## How to use this repo

Work through the modules in order; each builds on the last. Clone or download the repo, or simply read through the files directly on GitHub. Every module's exercises are designed to be done in a notebook, doc, or sketchpad of your own — there's no companion app required.

```bash
git clone https://github.com/NeuralStack-MS-Code/AI-Security-Engineering-for-Beginners.git
```
